# To-Do Application

To-Do application made for Back-end Development course.

## Quickstart

Firstly, clone the repository, then:

```bash
npm install
npm install --prefix server
node server/server
```

## Usage

1. Start the server:

   ```bash
   node server/server
   ```

   Backend default URL: http://localhost:3000

2. After starting the server, open the project file [client.html](client.html) in your browser.

## Features
 - Create an account and login
 - Insert tasks for each day of the week
 - Specify date, day of the week and schedule for each task
 - View, edit, organize and delete inserted tasks
